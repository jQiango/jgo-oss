## 🏗️ 技术架构

### 后端技术栈
```
Spring Boot 3.4.8
├── AWS SDK for Java 2.28.16 (S3客户端)
├── MyBatis-Plus 3.5.9 (ORM)
├── MySQL 8.0+ (数据库)
├── Lombok 1.18.30 (代码简化)
└── Hutool 5.8.25 (工具库)
```

### 前端技术栈
```
Vue 3 (CDN)
├── Tailwind CSS 3.x (样式)
├── Axios 1.x (HTTP客户端)
├── Bootstrap Icons 1.11.3 (图标)
├── Marked.js 11.1.1 (Markdown)
├── Highlight.js 11.9.0 (代码高亮)
└── Service Worker (PWA)
```

### 架构特点
- ✅ **分层架构**: Controller → Service → Mapper → Storage
- ✅ **安全设计**: 文件类型校验、路径遍历防护
- ✅ **缓存机制**: 内存+磁盘两层缓存
- ✅ **性能优化**: 虚拟滚动、懒加载、Service Worker
- ✅ **扩展性强**: 支持所有S3协议兼容存储

## 💡 技术亮点

### 1. 无构建前端
使用Vue 3 CDN模式，无需npm、webpack等构建工具：
- ✅ 修改代码立即生效
- ✅ 开发效率高
- ✅ 部署简单
- ✅ 学习成本低

### 2. 两层缓存机制
内存+磁盘双层缓存：
- ✅ 快速访问热数据
- ✅ 减少S3请求
- ✅ 降低带宽成本
- ✅ LRU自动淘汰

### 3. 丰富的文件预览
支持10+种文件类型预览：
- ✅ 图片（JPG、PNG、GIF、WEBP等）
- ✅ 视频（MP4、AVI、MKV等）
- ✅ 音频（MP3、WAV、OGG等）
- ✅ 文档（PDF、TXT、MD等）
- ✅ Office（Word、Excel、PPT）

### 4. PWA离线支持
Service Worker实现离线访问：
- ✅ 缓存静态资源
- ✅ 缓存API响应
- ✅ 离线可用
- ✅ 快速加载

### 5. 现代化UI
Vue 3 + Tailwind CSS：
- ✅ 响应式设计
- ✅ 暗色模式
- ✅ 流畅动画
- ✅ 优秀体验
